# 同意チェック追加プラグイン

ゲスト購入フォームとお問い合わせフォームにチェックボックスを追加し、チェックがなければ先へ進めないようにするプラグインです。

ゲスト購入フォームとお問い合わせフォームのいずれか、もしくは両方に追加するかは選択できます。

チェックボックスを自動で挿入する機能もあります。自動で挿入する場合はチェックボックスのラベルを管理画面で設定します。  
テンプレートをカスタマイズしていたり、独自のテンプレートを使用している場合には、自動挿入が正しく行われないことがあります。
その場合は自動挿入の機能を使用せず、「非会員購入情報入力」や「お問い合わせ(入力ページ)」のテンプレートを編集して、スニペットを挿入してください。  
挿入するスニペットは下記を参考にしてください。
```html
<div class="ec-checkbox{{ has_errors(form.user_policy_check) ? ' error' }}">
    <label>
        {{ form_widget(form.user_policy_check) }}
        <a class="ec-link" href="/help/agreement" target="_blank">利用規約</a>に同意する
    </label>
    {{ form_errors(form.user_policy_check) }}
</div>
```
お問い合わせフォームへスニペットを挿入する場合は、「お問い合わせ(確認ページ)」のテンプレートへも下記のスニペットを挿入してください。
```html
{{ form_widget(form.user_policy_check, { type : 'hidden' }) }}
```

以下に該当する場合、プラグインが正常に動作しない可能性があります。
* EC-CUBE本体をカスタマイズされている場合
* 他社製のプラグインを利用されている場合
* 対象バージョンではない場合

## インストールと有効化

1. 管理画面の「購入済プラグイン」画面上に表示されているプラグインの「インストール」ボタンをクリックしてインストール。
2. インストール済プラグインの「有効にする」をクリック。
